
# ⚖️ What is Elastic Load Balancing (ELB) in AWS?


> **Elastic Load Balancing (ELB)** is a managed AWS service that **automatically distributes incoming application or network traffic** across multiple targets (like EC2 instances, containers, IPs) in **one or more Availability Zones**, helping you **scale your application** and ensure **high availability** and **fault tolerance**.

Think of ELB as a **traffic manager** that ensures **no single server is overwhelmed**, and that **only healthy servers** receive traffic.
