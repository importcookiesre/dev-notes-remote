

> [!question] Define Indexes :
> Indexes are smaller referential tables that holds row references against the indexed value. Simply they are key value pairs where key is the indexed column value and value is the pointer to the actual record/row in the db.


> [!question] Types of indexing 
> 1. Primary : [[primary]]
> 2. clustered : [[clustered]]
> 3. secondary : [[secondary]]
